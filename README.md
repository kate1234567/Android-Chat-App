# Android Chat App

## Описание

Android Chat App — Android-приложение чата, разработанное на Kotlin.

Приложение поддерживает авторизацию, просмотр списка чатов, загрузку сообщений, отправку текстовых сообщений, отображение изображений и работу с кэшем при отсутствии сети.

Проект выполнен в рамках лабораторной работы по дисциплине «Мобильная разработка».

## Используемые технологии

- Kotlin
- Android SDK
- MVVM
- Dependency Injection
- Room
- WebSocket
- REST API
- Coroutines / Flow
- Jetpack Navigation

## Основные возможности

- экран входа;
- просмотр списка чатов;
- просмотр сообщений выбранного чата;
- отправка сообщений;
- отображение изображений;
- открытие изображения на отдельном экране;
- поддержка портретной и ландшафтной ориентации;
- сохранение выбранного чата при повороте экрана;
- кэширование ранее загруженных сообщений;
- работа с WebSocket;
- повторная авторизация при ошибке 401.

## Архитектура

Проект построен с использованием MVVM и разделён на несколько слоёв:

- `data/api` — работа с серверным API;
- `data/local` — локальное хранилище и кэш;
- `data/repository` — репозиторий данных;
- `data/websocket` — работа с WebSocket;
- `di` — настройка зависимостей;
- `ui/screens` — экраны приложения;
- `ui/viewmodel` — ViewModel-слой;
- `ui/navigation` — навигация между экранами.

## Запуск проекта

1. Открыть проект в Android Studio.
2. Дождаться синхронизации Gradle.
3. Указать необходимые параметры подключения к серверу, если они требуются.
4. Запустить приложение на эмуляторе или Android-устройстве.
